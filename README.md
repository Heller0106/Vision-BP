# Vision-BP

## Verschillende functionaliteiten uittesten

- **`c`**: Origineel kleurenbeeld.
- **`g`**: Grijswaarden analyse.
- **`e`**: Rode Randdetectie (Canny Edge Detection) met instelbare dikte.
- **`b`**: Blur (vervaging) effect voor ruisonderdrukking.
- **`m` / `n` / `o`**: Horizontaal, verticaal of volledig spiegelen.
- **`s`**: Screenshot maken (wordt opgeslagen in de map `/screenshots`).
- **`q`**: Programma veilig afsluiten.
